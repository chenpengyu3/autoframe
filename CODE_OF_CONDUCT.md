# Code of Conduct

AutoFrame follows the Contributor Covenant principles: be respectful, constructive, and welcoming.

Examples of encouraged behavior:

- Use inclusive language.
- Accept constructive feedback gracefully.
- Focus reviews on code, tests, design, and user impact.
- Help newcomers understand project conventions.

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks.
- Publishing private information without permission.
- Disruptive or bad-faith participation.

Maintainers may remove comments, reject contributions, or limit participation when behavior harms the project community.
