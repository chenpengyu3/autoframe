覆盖面从接口一路延伸到源码、数据库与运行时风险

测试范围被重新组织成六个可扫描能力域，让专业开发者快速判断风险入口，也让管理者和导师更容易理解质量覆盖。

API 功能矩阵

发现运行时端点、请求结构、CRUD 候选与协议兼容性，把接口行为转成可执行测试矩阵。

  • HEAD / OPTIONS
  • Accept 变体
  • 请求 schema

安全与 DAST 基线

覆盖认证绕过、SQL 注入、XSS 反射、路径遍历与敏感静态文件，让安全风险具备可追踪证据。

  • SQLi
  • XSS
  • Path traversal

性能与可靠性

输出延迟、吞吐、超时、降级与恢复信号，帮助团队区分业务失败、系统错误和容量风险。

  • P50 / P90 / P99
  • 吞吐量
  • 错误率

数据库结构矩阵

读取表、列、主外键、索引、唯一约束与字段类型，把真实 schema 转化成自动校验入口。

  • Tables
  • Indexes
  • Foreign keys

源码结构识别

识别 controller、service、repository、entity、事务、注入点和定时任务，减少手工补测试清单。

  • Spring Boot
  • Python
  • Manifests

生态工具就绪度

检测 Schemathesis、k6、ZAP、RESTler、Docker 等外部工具是否可接入成熟质量流程。

  • Schemathesis
  • k6
  • OWASP ZAP
